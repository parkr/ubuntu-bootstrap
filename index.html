    Ubuntu (Dev) Bootstrap

    Get your Ubuntu machine up to today's developer standards, quickly & easily.

    Install a module

    1. Ensure that you have a clone of this repo.
    2. cd into the clone
    3. Run bash [module] to execute the installation code
    4. Grab a cup of coffee and sip, enjoying the relaxation that this easy-to-use module-specific bootstrap script set has given you.
